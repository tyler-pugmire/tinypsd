# tinypsd

### FEATURES

#### Supported Formats

- [X] PSD
- [ ] PSB

#### Supported Compressions

- [X] Raw
- [X] RLE
- [ ] Zip without Prediction
- [ ] Zip with Prediction

#### Supported Color Modes

- [ ] Bitmap
- [X] Grayscale (no alpha support yet)
- [ ] Indexed
- [X] RGB
- [X] CMYK
- [ ] Multichannel
- [ ] Duotone
- [ ] Lab

#### Supported Depths

- [ ] 1-bit (Bitmap)
- [X] 8-bits (Grayscale, RGB, CMYK)
- [ ] 16-bits
- [ ] 32-bits
